# Bouyomi San YoutubeLive
Bouyomi San reads YouTubeLive chat comments with Web Speech API.

## How to use
```
$ pub get
$ pub serve
```
### api-key
https://console.cloud.google.com/apis

### channel-id
`https://www.youtube.com/channel/<here>`
